# Security

This repository is an assignment-style backend project and is not intended to run with real production credentials in its default configuration.

## Reporting

If you discover a security issue, please avoid opening a public issue with sensitive details. Instead, share a minimal reproduction and impact summary privately.

## Notes

- Do not commit `.env` files or real secrets (use `.env.example` as a template).
- Set `JWT_SECRET` to a long random value in any non-local deployment.
